  • LinkedIn Banner

    I made a LinkedIn banner that will hopefully attract more attention when someone is on my page. The banner will serve as a mini description of my skills by illustrating my core skill areas in web dev, embedded systems, game dev, and software development. It was created using Adobe Illustrator and was a fun one day project.

    Because LinkedIn utilizes a lighter theme, I chose a dark theme to help my illustration stand out. In addition, I wanted my skills to stick out, so I only used color when displaying a specific skill.

    I also put 2 hidden messages into this banner. Find them!
